# Todo Api Express

## 推奨開発環境

### VSCode RemoteDevelopment の場合

別リポジトリ ayataka0nk/todo-env の dev-api に用意。

### 他の場合

Dockerfile の dev ステージングから勝手にビルドして環境変数設定してよしなに。

## 環境変数

### ORMCONFIG_HOST

接続する DB のホスト名

### ORMCONFIG_PORT

接続する DB の使用ポート

### ORMCONFIG_USERNAME

接続する DB のユーザ名

### ORMCONFIG_PASSWORD

接続する DB のパスワード
